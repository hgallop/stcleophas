ASSETS FOLDER — Saint Cleophas Outreach Ministry
=================================================

Place the following files in this folder:

  logo.png
    The parish logo (Emmaus scene woodcut, provided as PNG).
    Recommended: save at 200×200 px or larger.
    The site references it as: assets/logo.png

The logo is displayed:
  - In the site header (inverted to white via CSS filter)
  - In the site footer (inverted to white, reduced opacity)

Once logo.png is in this folder, it will appear automatically
on all six pages without any code changes needed.

For team photos (Ministries page):
  Add an <img> tag inside the .team-photo div in ministries.html
  replacing the placeholder text span.
  Recommended size: 260×260 px, square crop.
